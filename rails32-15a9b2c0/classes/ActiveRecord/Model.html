Module ActiveRecord::Model

  • activerecord/lib/active_record/model.rb

ActiveRecord::Model can be included into a class to add Active Record persistence. This is an alternative to inheriting from ActiveRecord::Base. Example:

class Post
  include ActiveRecord::Model
end

def self.included(base)
  return if base.singleton_class < ClassMethods

  base.class_eval do
    extend ClassMethods
    Callbacks::Register.setup(self)
    initialize_generated_modules unless self == Base
  end
end

Module ERB::DefMethod

  • lib/erb.rb

Utility module to define eRuby script as instance method.

Example

example.rhtml: 

  <% for item in @items %>
  <b><%= item %></b>
  <% end %>

example.rb: 

  require 'erb'
  class MyClass
    extend ERB::DefMethod
    def_erb_method('render()', 'example.rhtml')
    def initialize(items)
      @items = items
    end
  end
  print MyClass.new([10,20,30]).render()

result: 

  <b>10</b>

  <b>20</b>

  <b>30</b>

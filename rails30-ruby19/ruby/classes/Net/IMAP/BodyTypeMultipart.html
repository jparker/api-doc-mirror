Class Net::IMAP::BodyTypeMultipart < Struct.new(:media_type, :subtype, :parts, :param, :disposition, :language, :extension)

  • lib/net/imap.rb

Net::IMAP::BodyTypeMultipart represents multipart body structures of messages.

Fields:

media_type:Returns the content media type name as defined in [MIME-IMB].
subtype:Returns the content subtype name as defined in [MIME-IMB].
parts:Returns multiple parts.
param:Returns a hash that represents parameters as defined in [MIME-IMB].
disposition:Returns a Net::IMAP::ContentDisposition object giving the content disposition.
language:Returns a string or an array of strings giving the body language value as defined in [LANGUAGE-TAGS].
extension:Returns extension data.
multipart?:Returns true.

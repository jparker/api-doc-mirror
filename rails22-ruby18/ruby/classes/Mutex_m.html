Module Mutex_m

  • lib/mutex_m.rb

Usage

Extend an object and use it like a Mutex object: 

  require "mutex_m.rb"
  obj = Object.new
  obj.extend Mutex_m
  # ...

Or, include Mutex_m in a class to have its instances behave like a Mutex object: 

  class Foo
    include Mutex_m
    # ...
  end

  obj = Foo.new

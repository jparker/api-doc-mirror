advance(options)

Provides precise Date calculations for years, months, and days. The options parameter takes a hash with any of these keys: :years, :months, :weeks, :days.

Source: show | on GitHub

 95:           d = self
 96:           d = d >> options.delete(:years) * 12 if options[:years]
 97:           d = d >> options.delete(:months)     if options[:months]

 99:           d = d +  options.delete(:days)       if options[:days]

change(options)

Returns a new Date where one or more of the elements have been changed according to the options parameter.

Examples: 

  Date.new(2007, 5, 12).change(:day => 1)                  # => Date.new(2007, 5, 1)
  Date.new(2007, 5, 12).change(:year => 2005, :month => 1) # => Date.new(2005, 1, 12)
